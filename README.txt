DESCRIPTION:
-----------
A modern, standalone Windows application to convert PNG images to ICO (icon) format.
No Python installation required!

FEATURES:
---------
✓ Simple and modern GUI interface
✓ Multiple icon sizes (16x16 to 256x256)
✓ Generate all sizes at once
✓ Auto-detect new PNG files in Downloads folder
✓ Auto-delete original PNG option
✓ Custom output folder selection
✓ Live image preview
✓ Quick access to Flaticon for icon downloads
✓ Standalone executable - no dependencies!

INSTALLATION & USAGE:
--------------------
1. Double-click "PngToIco.exe" to launch the application
2. Click "Select PNG Image" button OR simply download a PNG to your Downloads folder
3. Choose the desired ICO size from the dropdown menu
4. (Optional) Check "Generate all common sizes" for multi-size ICO file
5. Click "Convert to ICO" button
6. Your ICO file will be saved to the selected output folder (default: Downloads)

That's it! No installation, no setup, just run and convert!
